# Gravity

Gravity is a solar system simulation.

Keyboard controls :

wheel_up : Zoom in

wheel_down : Zoom out

\+ : increase time speed by 1 year/s

\- : decrease time speed by 1 year/s

Ctl + (+) : increase time speed by 10 year/s

Ctl + (-) : decrease time speed by 10 year/s

space : pause the time of the simulation (planets stop moving)

f : unlock freeView to navigate in 3D with :
  - arrow_up : move forward
  - arrow_down : move backward
  - arrow_left : move left
  - arrow_right : move right
  - mouse : free looking direction 

When not in freeView :
  - arrow_right : switch planet's focus (Next)
  - arrow_left : switch planet's focus (Previous)
  - R : reset focus to Sun
